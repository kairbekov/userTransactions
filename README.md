FastAPI simple project of User Transactions

How to start:
1. Clone project
```console
git clone https://github.com/kairbekov/userTransactions.git
```
2. go to project directory
```console
cd userTransactions/
```
3. build images with docker-compose
```console
docker-compose up -d --build
```
